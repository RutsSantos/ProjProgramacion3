# **Estructura del proyecto**

## El proyecto consta con las siguientes carpetas:

* *Inventario*:
    * `Img`: donde se almacenan todas los iconos que se utilizaron
    * `Consultas`: contiene todos los windows form para consultar las entidades del proyecto
    * `DataSets`: contiene las configuraciones de como se verán las informaciones de la base de datos.
    * `Ejercicios`: donde están alamcenados todas las practicas.
    * `Mantenimiento`: contiene todos los windows form para interacturar con las entidades del proyecto, con los CRUD para las tablas de la base de datos.
    * `Procesos`: contiene el form con el cual se realiza el proceso de venta de productos a los clientes
    * `Reportes`: windows form que contiene los reportes generados desde las consultas.
    * `Backup BD`: carpeta en donde se alamcena todos los archivos relacionados a la base de datos
    * `Reports`: el view que muestra los reportes

* *Utilidades*: 
    * Diferentes algoritmo para el funcionamiento del proyecto
